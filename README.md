# wukong-invite-helper

中文 | [English](./README_EN.md)

自动获取悟空邀请码的工具。实时监控官方接口，识别新图片中的邀请码，自动复制到剪贴板并填入悟空 App。

## 使用前准备

1. 下载 [悟空 App](https://wukong.dingtalk.com/)
2. 用钉钉扫码登录
3. 停留在邀请码输入页面

## 一键启动（推荐）

双击即用，自动完成环境配置。首次启动约 1-2 分钟，之后秒开。

| 系统 | 操作 |
|------|------|
| **Windows** | 双击 `start.bat` |
| **macOS** | 双击 `start.command`（首次需右键 → 打开） |

启动后浏览器会自动打开 Web UI 界面。

## Web UI 功能

访问 `http://127.0.0.1:8787`（启动后自动打开）

- **开始监听**：自动轮询新邀请码
- **手动重试**：立即重新识别当前图片
- **复制邀请码**：一键复制到剪贴板
- **清空记录**：删除已处理的图片 ID，允许重新识别

## macOS 用户注意

如需「自动填入悟空 App」功能，需开启辅助功能权限：

`系统设置 → 隐私与安全性 → 辅助功能` → 勾选你的终端 App

## 常见问题

| 问题 | 解决方法 |
|------|----------|
| 没有识别出邀请码 | 点击「清空当前 seen_id」后重试 |
| 邀请码识别错误 | 点击「手动重试」再次识别 |
| macOS 自动填入无效 | 检查终端是否有辅助功能权限 |
| Windows 自动填入无效 | 确认悟空 App 窗口已打开 |

## 配置说明

Web UI 界面支持以下配置：

| 配置项 | 默认值 | 说明 |
|--------|--------|------|
| 轮询间隔 | 1 秒 | 检查新图片的频率 |
| 最长运行 | 300 秒 | 自动停止时间 |
| 复制到剪贴板 | 开启 | 识别成功后自动复制 |
| 提示音 | 开启 | 识别成功后播放声音 |
| 自动填入 | 开启 | 自动填入悟空 App |
| 自动提交 | 开启 | 填入后自动回车 |
